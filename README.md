# ODT Technology Radar

> Inspired by Thoughtworks Technology Radar

เครื่องมือที่ใช้คือ [Obsidian](https://obsidian.md/)

## วิธีการเขียน

- ค้นหาก่อนเสมอว่ามีเอกสารอยู่หรือไม่ ให้ใช้ Omnisearch ที่เมนูทางซ้ายมือของ Obsidian หรือกด `Cmd + Shift + O`
- ให้เขียน 1 เอกสาร สำหรับ 1 Technology เพื่อที่เวลา Link กลับเข้ามาในแต่ละวง หรือในแต่ละเอกสารรวบรวม (Volume) จะได้ดูจาก Graph view ได้ง่าย
- ตอนที่ช่วยกันเขียนอาจจะมี Conflict เกิดขึ้นที่ไฟล์ `.obsidian/workspace.json` ได้ ไม่ต้องสนใจมาก เพราะว่าเดี๋ยว Obsidian บนเครื่องแต่ละคนจะ Override ไฟล์นั้นอีกที
